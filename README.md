# Iris Flower Classification

> A machine learning project demonstrating classification algorithms using the famous Iris dataset

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Scikit-learn](https://img.shields.io/badge/scikit--learn-0.24+-orange.svg)](https://scikit-learn.org/)
[![Jupyter Notebook](https://img.shields.io/badge/jupyter-notebook-orange.svg)](https://jupyter.org/)

## 📋 Overview

This project implements machine learning classification algorithms to predict iris flower species based on physical measurements. It demonstrates fundamental ML concepts including data preprocessing, model training, and evaluation using the classic Iris dataset.

## ✨ Key Features

- 📊 **Data Exploration** - Statistical analysis and visualization of iris features
- 🤖 **Multiple Algorithms** - Comparison of different classification models
- 📈 **Performance Metrics** - Accuracy, precision, recall, and F1-score evaluation
- 🎯 **Model Selection** - Best model identification and cross-validation
- 📉 **Visualizations** - Feature distributions and decision boundaries

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- Jupyter Notebook
- scikit-learn
- pandas
- matplotlib
- seaborn

### Installation

```bash
# Clone the repository
git clone https://github.com/Shivansh-mishraji/Iris-Flower-Classification.git
cd Iris-Flower-Classification

# Install dependencies
pip install scikit-learn pandas matplotlib seaborn jupyter

# Start Jupyter
jupyter notebook
```

## 📊 Dataset

- **Source**: UCI Machine Learning Repository
- **Samples**: 150 iris flowers
- **Features**: 4 (sepal length, sepal width, petal length, petal width)
- **Classes**: 3 (Iris setosa, Iris versicolor, Iris virginica)
